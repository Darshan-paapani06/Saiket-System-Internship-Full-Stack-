### ⚡ Task 5 — Database Integration

> **Goal:** Connect the REST API to MySQL. Store and retrieve user data using SQL queries via API endpoints.

<details>
<summary><b>📋 View Details</b></summary>

<br/>

**Database Setup:**
```sql
CREATE DATABASE IF NOT EXISTS saiket_internship;
USE saiket_internship;

CREATE TABLE IF NOT EXISTS users (
  id         INT AUTO_INCREMENT PRIMARY KEY,
  name       VARCHAR(100) NOT NULL,
  email      VARCHAR(150) NOT NULL UNIQUE,
  age        INT NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  CONSTRAINT chk_age CHECK (age BETWEEN 1 AND 120)
);
```

**All 5 CRUD endpoints** now connected to MySQL using `mysql2/promise`:

```
GET    /users      → SELECT * FROM users ORDER BY id DESC
GET    /users/:id  → SELECT * FROM users WHERE id = ?
POST   /users      → INSERT INTO users (name, email, age) VALUES (?, ?, ?)
PUT    /users/:id  → UPDATE users SET name=?, email=?, age=? WHERE id=?
DELETE /users/:id  → DELETE FROM users WHERE id=?
```

**Skills Used:** `Basic SQL` `MySQL` `Database Connection` `mysql2` `API Integration` `Express.js`

**Files:** `task5-db-integration/index.js` · `task5-db-integration/schema.sql` · `task5-db-integration/package.json`

**Run:**
```bash
# Step 1 — MySQL setup
mysql -u root -p
source path/to/task5-db-integration/schema.sql;

# Step 2 — Update password in index.js line 9
# password: "your_mysql_password"

# Step 3 — Start server
cd task5-db-integration
npm install
node index.js
# → http://localhost:3000
```

### ⚡ Task 6 — Full Stack Application

> **Goal:** Combine front-end and back-end into a complete User Management System with login authentication.

<details>
<summary><b>📋 View Details</b></summary>

<br/>

**Architecture:**
```
┌─────────────────────┐        HTTP + JWT        ┌──────────────────────┐
│   Frontend          │ ◄──────────────────────► │   Backend            │
│   client/index.html │                           │   server/index.js    │
│   Port: 5500        │                           │   Port: 5000         │
└─────────────────────┘                           └──────────┬───────────┘
                                                             │  mysql2
                                                  ┌──────────▼───────────┐
                                                  │   MySQL Database      │
                                                  │   saiket_fullstack    │
                                                  └──────────────────────┘
```

**Auth Endpoints:**
| Method | Route | Description |
|--------|-------|-------------|
| `POST` | `/api/auth/register` | Register with name, email, age, password |
| `POST` | `/api/auth/login` | Login → returns JWT token |

**Protected User Endpoints (JWT required):**
| Method | Route | Description |
|--------|-------|-------------|
| `GET` | `/api/users` | Fetch all users |
| `GET` | `/api/users/:id` | Fetch single user |
| `PUT` | `/api/users/:id` | Update user profile |
| `DELETE` | `/api/users/:id` | Delete user |

**Frontend Pages:**
- 🔐 **Login / Register** — tab-switched auth form with full validation
- 📊 **Dashboard** — stats cards (Total Users, Role)
- 📋 **User Table** — view all users with search
- ➕ **Add User Modal** — form with validation
- ✏️ **Edit User Modal** — pre-filled form
- 🗑️ **Delete Confirm Modal** — safety confirmation
- 🔔 **Toast Notifications** — success / error feedback
- 💾 **JWT persisted** — stays logged in on page refresh

**Skills Used:** `Full Stack Development` `JWT Auth` `bcryptjs` `CORS` `MySQL` `Express` `Vanilla JS` `Fetch API`

**Files:**
```
task6-fullstack/
├── server/index.js          ← Express + MySQL + JWT backend
├── server/package.json
├── server/schema.sql        ← MySQL table with password column
└── client/index.html        ← Complete frontend (no frameworks)
```

**Run:**
